Components and Modifiers:

What is a component?

Component is a CSS class which can be used anywhere in the code.

In polyflex there're a few components:

  • row - single row of items (flexbox),
  • rows - multiple rows of items (flexbox),
  • col - single column of items (flexbox),
  • cols - multiple columns of items (flexbox),
  • item - single item in row or column,
  • break - go to the next row or column.

What is a modifier?

Modifier is a special CSS class which can be used only with certain components. Modifier name is prepended with dash symbol or breakpoint code (e.g. xs-*).

In polyflex there're following modifiers:

  • basis - controls the width of items,
  • scroll - adds a scrollbar for overflowing items,
  • grow-* - defines the ability for an item to grow if necessary,
  • no-grow - block items ability to grow,
  • shrink-* - defines the ability for an item to shrink if necessary,
  • no-shrink - block items ability to shrink,
  • cols-* - align all items in a single col,
  • rows-* - align all items in a single row,
  • x-* - control items x position in the flexbox,
  • y-* - control items y position in the flexbox,
  • order-* - specify the item position in a flexbox.
